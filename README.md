# QR code generator

Enter a valid and get the QR, which can be downloaded as SVG or PNG.

Built using:

-  Vue
-  Vuetify
-  qrtag.net API
-  Vuex
-  SCSS

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development
