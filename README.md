# QR code generator

Enter a valid, press the button(or enter key) and get the QR(almost instantly). The code can then be downloaded as SVG or PNG.

Hosted on Vercel at [GetQR](https://getqr.vercel.app/)

Built using:

-  Vue
-  Vuetify
-  qrtag.net API
-  Vuex
-  SCSS

## Project setup

```
yarn install
```
